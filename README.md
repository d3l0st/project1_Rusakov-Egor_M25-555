# Лабиринт Сокровищ

Текстовая приключенческая игра в жанре rogue-like, где игрок исследует таинственный лабиринт, решает загадки, собирает предметы и ищет легендарные сокровища.

## Особенности

- **Исследование лабиринта** - 8 уникальных комнат с разными тематиками
- **Система загадок** - интеллектуальные головоломки с альтернативными ответами
- **Инвентарь и предметы** - сбор и использование различных артефактов
- **Случайные события** - непредсказуемые встречи и находки
- **Ловушки и опасности** - будьте осторожны в темных коридорах
- **Множественные концовки** - найдите сокровище или попробуйте взломать замок

## Установка и запуск

### Предварительные требования
- Python 3.10 или выше
- Poetry (менеджер зависимостей)

### Установка
```bash
# Клонируйте репозиторий
git clone <ваш-репозиторий>
cd project1_Rusakov-Egor_M25-555

# Установите зависимости
poetry install

# Или используйте Makefile
make install
```
### Запуск
```bash
# Запуск используя poetry
poetry run project

# Или используйте Makefile
make project
```
